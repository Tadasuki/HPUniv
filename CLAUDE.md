## Project

Hugo blog using the [Reimu theme](https://github.com/D-Sketon/hugo-theme-reimu).

## Build & Serve

```bash
hugo serve        # local dev server
hugo              # build to public/
```

## Structure

- `hugo.toml` — site configuration
- `config/_default/params.yml` — theme configuration
- `content/post/` — blog posts
- `data/covers.yml` — post covers
